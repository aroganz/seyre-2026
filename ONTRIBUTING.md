📝 How to Provide Feedback for SeyRe Global
To keep our project organized, we are using GitHub Issues for all content updates and feedback. This ensures nothing gets lost.

1. How to Report a Change
Log into your GitHub account and go to our project link.

Click the "Issues" tab at the top.

Click the green "New Issue" button.

2. Best Way to Describe Your Input
When you create a new issue, please use a clear Title and Description:

Title: Use the page name (e.g., [Home Page] Update Partner Logo or [Team Page] Change Bio for CEO).

Description: * Where: (Paste the URL of the page).

Current Text: (What is currently there).

New Text: (What you want it to say).

Attachment: You can drag and drop images directly into the box if you need to show us a screenshot.

3. Reviewing our Progress
Once you submit an issue, I will label it (e.g., "Content Update" or "Bug").

You will get an email notification when I comment on it or mark it as "Closed" (meaning the change is live).
